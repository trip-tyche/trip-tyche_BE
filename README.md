## 📸 About 트립티케

- 트립티케는 여행 사진 속 메타데이터를 활용해 자동으로 여행 경로를 만들어주는 서비스입니다.
- 사진을 업로드하면 위치/날짜 정보가 자동으로 추출되어 구글 맵에 원하는 방식으로 표시됩니다.
- 이미지에 위치 정보가 없다면, 간단히 지도에서 위치를 직접 선택할 수도 있습니다.

## 🧷 주요 기능

| **⏏️ 자동 위치 정보 추출**                                                                 | **🚶 여행 경로 애니메이션**                                                       |
|:--------------------------------------------------------------------------------------:|:-------------------------------------------------------------------------------:|
| ![위치추출](https://github.com/trip-tyche/trip-tyche_FE/blob/main/docs/images/image-upload.gif?raw=true) | ![타임라인](https://github.com/trip-tyche/trip-tyche_FE/blob/main/docs/images/timeline.gif?raw=true) |
| 메타데이터(위치, 날짜) 추출 및 리사이징 WebP 변환으로 이미지 최적화           | 군집화된 여행 경로를 따라 움직이는 캐릭터 애니메이션 제공                   |
| **🎞️ 위치별 이미지 슬라이드쇼**                                                     | **📅 날짜별 이미지 갤러리**                                                      |
| ![위치별 이미지](https://github.com/trip-tyche/trip-tyche_FE/blob/main/docs/images/image-by-pinpoint.gif?raw=true) | ![날짜별 이미지](https://github.com/trip-tyche/trip-tyche_FE/blob/main/docs/images/image-by-date.gif?raw=true) |
| 핀포인트에 해당하는 이미지 슬라이드쇼                              | 시간순으로 정렬된 날짜별 이미지 갤러리                                      |
| **📍 동적 클러스터링**                                                             |
| ![클러스터링](https://github.com/trip-tyche/trip-tyche_FE/blob/main/docs/images/clustering.gif?raw=true) |
| Google Maps Clusterer를 활용한 줌 레벨 기반 동적 이미지 클러스터링            |
